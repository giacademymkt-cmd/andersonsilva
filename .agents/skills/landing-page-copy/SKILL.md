# Instruções
O copy deve evitar clichês esportivos.
Priorizar palavras como: Constância, Energia, Acompanhamento, Autonomia, Equilíbrio.
Evitar palavras como: Projeto Verão, Secar, Dieta Restrita, No Pain No Gain.
